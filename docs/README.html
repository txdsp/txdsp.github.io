txdsp.net¶

This webpage contains supplemental material for EE 445S real-time digital signal processing laboratory

Organization¶

The source materials consist of MyST markdown files (.md) and Jupyter notebooks (.ipynb) which are built into a webpage using Jupyter Book.

  • MyST cheat sheet

  • Math and equations

  • Images and figures

  • Notebooks

  • Cells and metadata

  • Table of contents

Contribute¶

If you would like to be added as a contributor to this repository, contact Dan Jacobellis

Building and hosting¶

The build script compiles the source materials into a webpage located in the docs folder. Once changes are committed to this repository, they are automatically published using GitHub pages to txdsp.net
