Installation and getting started

DSP theory and algorithms can be studied from a purely mathematical perspective with minimal computing or programming background. However, a more holistic perspective acknowledges tradeoffs between signal quality and implementation complexity.

Although some DSP systems are implemented in assembly or hardware description languages, it is more common to use a low level programming language like C. high level languages are often used for prototyping and simulation of DSP algorithms.

If you are not comfortable with programming, don’t worry! Computing and DSP are complementary; a great way to demystify DSP algorithms is by implementing them in software, and great way to learn software design patterns is by implementing DSP algorithms.

In this course, we will develop the embedded implementation in C. Prototyping and simulation can be done using either Julia, Python, or MATLAB.

Package management¶

Many of the most frustrating aspects of programming and software development can be avoided by using a package manager, both for your operating system and for your particular language.

  • On Windows, the package manager can be accessed using the winget command in powershell.

  • Mac OS, Homebrew is the most popular package manager.

  • Python’s package manager pip is a standalone program which can be accessed using either the command pip or pip3 depending on your installation.

  • Julia’s package manager Pkg can be accessed by pressing the ] key in the Julia REPL

Jupyter¶

Interactive programming can be extremely helpful for simulation and prototyping. Project Jupyter (portmanteau of Julia, Python, R) allows for several languages to be developed interactively using notebooks.

The easiest way to get started is by first installing Python and then using pip to install Jupyter Lab

pip install jupyterlab

Then, you can run it using the jupyter-lab command.

Python¶

Python is currently the most popular general purpose programming language. Use in embedded and real-time systems is possible via micropython.

Installation and getting started

You can install Python 3 using your operating system’s package manager.

winget install Python.Python.3

  • Homebrew and python

Warning

Many operating systems will have Python 2 pre-installed. Depending on your operating system and installation process, the python and pip commands might point to Python 2 while python3 and pip3 commands point the python 3. You may want to create or modify any symbolic links to the python and pip commands after installation to avoid confusion or modify path environemt variables

A great way to write Python code for simulation and prototyping is with Jupyter Lab.

Julia¶

Julia is one of the fastest growing languages. It is increasingly used in education and industry, especially for high-performance computing and machine learning applications.

Installation and getting started

You can install Julia using your operating system’s package manager.

winget install Julialang.Julia

brew install --cask julia

Note

The official Julia language organization also provides installers, platform-specific installation instructions, and several tutorials.

MATLAB¶

MATLAB is commercial software. Many textbooks, especially in the field of signal processing, use MATLAB since it offers a set of interoperable, application-specific toolboxes for everything from filter design to embedded code generation.
